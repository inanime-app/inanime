## Features

### Core Features
- **Multi-Language Support**: Watch anime in Sub (subbed), Dub (dubbed), and Hindi dubbed versions
- **Offline Downloads**: Download episodes to watch offline anytime
- **Built-in Player**: Advanced video player with playback controls
- **Search & Filter**: Find anime by title, genre, year, status
- **Favorites**: Save your favorite anime for quick access
- **Watch History**: Track your watching progress

### Social Features
- **Community Chat**: Chat with fellow anime fans in real-time
- **Personal Messaging**: Send private messages to friends
- **User Profiles**: Create and customize your anime profile
- **Rankings**: View top-ranked anime by popularity

### Gamification
- **Quests**: Complete quests to earn rewards

## Installation

### Requirements
- Android 5.0 (Lollipop) or higher
- Internet connection for streaming
- Storage permission for downloads

### **Smart & Connected**
*   **Auto-Updater**: Automatically checks for updates via our custom API.
*   **No Account Required**: Start instantly with no sign-ups.
*   **Offline Access**: Save episodes locally for convenience.

###  **Beautiful Design**
*   **Modern UI/UX**: Sleek dark-themed interface with a clean layout.
*   **Personalization**: Manage favorites and continue watching easily.
